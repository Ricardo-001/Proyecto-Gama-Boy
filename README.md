# Proyecto Nintedo Game Boy con Html Css y Emulator.js

## Tecnologías y herramientas utilizadas 
- Html
- Css 
- Emulator.js

## Detalles sobre este proyecto
Este es un proyecto personal untilizando Html Css y Emulator.Js para crear la replica de una Nintedo Game Boy color en donde es posible jugar el videojuego "Pokemon Gold edition"

La estructura del proyecto esta creada en Html y el estilo en Css, Emulator.js es una herramienta Creada en JavaScript diseñada para emular consolas de videojuegos retro en navegadores web
